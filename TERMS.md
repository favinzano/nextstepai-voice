# Términos De Uso

Última actualización: 6 de junio de 2026.

1. NextStepAI Voice es una herramienta de asistencia para dictado y transcripción.
2. El usuario es responsable de revisar el texto antes de enviarlo, publicarlo o utilizarlo en decisiones importantes.
3. La aplicación puede producir errores, omisiones o interpretaciones incorrectas.
4. No debe utilizarse como única fuente para transcripciones médicas, legales, financieras, de emergencia o de seguridad.
5. El usuario debe contar con autorización para grabar o procesar voces de terceros conforme a las leyes aplicables.
6. La aplicación se entrega sin garantía de disponibilidad, precisión absoluta o adecuación a un propósito específico.
7. Los modelos y componentes de terceros mantienen sus términos y licencias originales.
