# NextStepAI Voice 1.0.0

Primer release candidato para Windows x64.

## Funciones

- Dictado desde cualquier aplicación mediante atajo global.
- Pegado automático sin perder el foco.
- Whisper Base y Whisper Small ejecutados localmente.
- Limpieza de muletillas, espacios, emails, URLs y estructura.
- Diccionario e historial locales.
- Overlay de grabación privado.
- Reparación de caché de modelos.

## Privacidad

No incluye telemetría. El audio y la inferencia permanecen en el equipo. Se requiere internet únicamente para descargar los modelos por primera vez.

## Artefacto

- Instalador: `NextStepAI-Voice-Setup-1.0.0-x64.exe`
- Plataforma: Windows x64
- Firma: pendiente de certificado Authenticode

## Limitaciones Conocidas

- La precisión depende de voz, micrófono, ruido e idioma.
- El usuario debe revisar textos antes de utilizarlos.
- La versión pública estable requiere firma de código y completar la matriz humana de aceptación.
