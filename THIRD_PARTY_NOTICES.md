# Avisos De Terceros

NextStepAI Voice utiliza componentes de terceros. Este resumen no sustituye los textos completos incluidos por cada proyecto.

| Componente | Uso | Licencia |
|---|---|---|
| Electron | Aplicación de escritorio | MIT |
| Electron Builder | Empaquetado e instalador | MIT |
| esbuild | Compilación del renderer | MIT |
| Transformers.js | Ejecución local de modelos | Apache-2.0 |
| ONNX Runtime | Inferencia local | MIT |
| OpenAI Whisper | Arquitectura y modelos de transcripción | MIT |
| Whisper Base / Small de ONNX Community | Modelos ONNX cuantizados | Licencias indicadas por sus repositorios |
| Geist | Tipografía de interfaz | SIL Open Font License 1.1 |
| DM Serif Display | Tipografía editorial | SIL Open Font License 1.1 |

Los textos completos de licencia de las dependencias instaladas pueden consultarse en sus paquetes dentro de `node_modules` durante desarrollo y en las fuentes oficiales de cada proyecto.
